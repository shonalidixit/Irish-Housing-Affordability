# Irish Housing Affordability Analysis

This project analyses housing affordability trends in Ireland by examining the relationship between average rent, income, and affordability over time. Using publicly available data from the Central Statistics Office (CSO), the project aims to understand whether income growth has kept pace with rising housing costs across Irish cities.

##  Project Overview
Housing affordability has become a significant economic and social challenge in Ireland. This analysis focuses on:
- Long-term rent trends across Irish cities
- Comparison between rent growth and average earnings
- Changes in housing affordability over time
- Regional differences in rental costs

The analysis is supported by data preprocessing in Python and visualised using an interactive Power BI dashboard.

##  Tools & Technologies
- **Python** (Pandas, NumPy) – data cleaning and feature engineering  
- **Power BI** – dashboard creation and visual analysis  
- **Jupyter Notebook** – exploratory data analysis
